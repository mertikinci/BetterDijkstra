--> Maze-Puzzle-master isimli dosya ve altındakiler tamamen arayüzü oluşturma amaçlı olarak oluşturulmuş html ve css dosyalarıdır.

--> Maze-Puzzle-Solver-master isimli dosya ve altındaki dosyalarda ise temel algoritma kodlarını bulmak mümkündür.


Çalıştırmak için -->> önce repoyu bilgisayarınıza indirin ve Maze-Puzzle-Solver-master isimli klasörü build edin. Build işlemi tamamlandıktan sonra MazepuzzleApplication isimli class'ı run ederek algoritmayı çalışır hale getiriyoruz. Son olarak yapılması gereken işlem Maze-Puzzle-master klasörünün altında index.html isimli dosyayı çalıştırmak. Springboot çalışmaya başladıktan sonra herhangi bir tarayıcıya index.html dosyasını sürükleyip bırakmanız projenin çalışması için yeterli olacaktır.
